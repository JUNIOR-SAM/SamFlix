About SamEats

SamEats is a recipe explorer built by Samuel for the SQI "Show Your Work" project. It uses TheMealDB public API to fetch meal data. This demo project implements a simple authentication flow (localStorage-based), favorites per user, and a responsive UI built with Tailwind CSS.

Tech Stack

  • HTML / Tailwind CSS
  • Vanilla JavaScript (fetch API)
  • TheMealDB (https://themealdb.com)
  • localStorage for simple persistence
